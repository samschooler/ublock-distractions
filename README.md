# ublock-distractions
A filter list of visual element distractions for myself. Reddit, IG, Youtube, etc.

## Filters
- Reddit: Unusable unless accessing a thread directly. Good for recommendations, code help, etc. But no browsing.
- Youtube: No recs or comments. Just the video and descriptions.
